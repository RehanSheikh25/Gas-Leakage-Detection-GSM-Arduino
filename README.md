# 🔥 Gas Leakage Detection System using GSM & Arduino

## 📌 Overview
This project detects gas leakage using a gas sensor and sends an alert SMS using a GSM module. It helps prevent accidents by providing early warnings.

## ⚙️ Components Used
- Arduino Uno
- MQ Gas Sensor (MQ-135)
- GSM Module (SIM800L)
- Buzzer
- Jumper Wires
- Power Supply

## 🔧 Working Principle
- Gas sensor continuously monitors gas levels
- When gas exceeds threshold:
  - Buzzer activates
  - GSM module sends SMS & CALL alert
- System ensures real-time safety monitoring

## 🚀 Features
- Real-time gas detection
- Instant SMS & CALL alert system
- Low-cost and efficient
- Easy to install
